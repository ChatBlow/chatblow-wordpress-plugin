# ChatBlow WordPress Plugin

The official ChatBlow WordPress plugin to embed the multi channel customer support platform on your site.

ChatBlow is a powerful AI trained multi channel customer support platform that helps you engage with your community. It supports Facebook page comment and message auto reply, AI chatbot, unlimited live chat and support ticket with human agent.

## Features
* **Easy Integration:** No coding required. Simply install, configure, and your AI assistant is live.
* **Positioning:** Choose exactly where you want the chat widget to appear (bottom-right, bottom-left, etc.).
* **User Tracking (Optional):** Automatically pass logged-in WordPress user metadata (Name, Email) to ChatBlow to identify your users in chat threads.
* **Enable/Disable Toggle:** Quickly toggle the widget on or off directly from your WP admin dashboard.

## Privacy & External Service
This plugin relies on the ChatBlow API (`https://chatblow.com`) to verify your registration status and load the interactive chat widget script (`https://chatblow.com/static/assistant.js`). 

* If enabled, the plugin loads the assistant script from our servers. 
* If the "Track User" setting is enabled, logged-in user details (Name, Email, ID) are securely passed to the ChatBlow widget to enhance the personalized chat experience.

For more information, please refer to our [Privacy Policy](https://chatblow.com/privacy-policy) and [Terms of Service](https://chatblow.com/terms-and-conditions).

## Installation
1. Download the [latest release](https://github.com/ChatBlow/chatblow-wordpress-plugin/releases).
2. Upload the `chatblow` directory to your `/wp-content/plugins/` directory, or install the zip via the WP Admin Panel.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Navigate to **ChatBlow** in the admin sidebar.
5. Ensure your domain is registered at [ChatBlow.com](https://chatblow.com).
6. Configure your widget position and user tracking settings.
